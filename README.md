# My Portfolio Website

## About
This is my personal portfolio website showcasing my work as a Computer Science student.

## Features
- ✅ Responsive design for all devices
- ✅ 5 languages (English, Amharic, Oromo, Somali, Tigrinya)
- ✅ 4 color themes (Dark, Light, Blue, Green)
- ✅ Contact form with database storage
- ✅ Admin panel to view messages
- ✅ Udacity certificates with verification links

## Technologies Used
- HTML5, CSS3, JavaScript
- PHP for backend
- MySQL for database
